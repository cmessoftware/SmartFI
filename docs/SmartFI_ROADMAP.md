SmartFI V1 es un asistente inteligente para gestión de finanzas personales. Te permite

 - Gestión  de gastos
 - Gestión de ingresos
 - Gestión de tarjetas de credito (In progress)
 - Gestión de deudas (In progress)
 - Alertas de vencimientos
 - Dashboard simples
 - Soporta carga masivas desde csv, download de resumens en csv.
 - Se conecta a google shseet para exportar la info.
 - Aperturea y cierre de mes para presupeustos y gastos/ingresos (pendiente: proyeccioens de gastos con tarjeta de cédito)
