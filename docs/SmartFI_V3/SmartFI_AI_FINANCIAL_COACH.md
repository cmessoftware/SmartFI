SMART FI Version 3

 SmartFi V3. New features:
  
 - AI Chatbot using knowledge base for user support.
 - More elaborate, interactive and customizable dashboards from the front end.
 - Alerts sent to WhatsApp and Telegram
 - Extension of LLM engine for suggestions and improvements of expenses based on savings goals, investment, reduction of impulse spending etc. In that case it can be extended with suggestions for documentation, videos and related channels.
